# CURSO DE PROGRAMAÇÃO

Este repositório contém as aulas e exercícios de um curso introdutório de programação em JavaScript. Cada pasta representa uma aula com exemplos práticos e exercícios para aprender os conceitos básicos da linguagem.

## Estrutura do Projeto

- **AULA 02/**: Introdução ao console e variáveis básicas.
- **AULA 04/**: Coleta de dados com variáveis (let, const, var) e manipulação do DOM.
- **AULA 05/**: Operadores Aritméticos
- **AULA 06/**: Operadores de Comparação e Lógicos
- **AULA 07/**: Condicionais
- **AULA 08/**: Switch
- **AULA 09/**: Funções
- **AULA 10/**: Funções
- **AULA 11/**: Funções Anônimas e Arrow Functions
- **AULA 12/**: Arrays
- **AULA 13/**: JavaScript: JS
- **AULA 14/**: Variáveis
- **AULA 15/**: For Each e Map
- **AULA 16/**: Trabalhando com objetos em JavaScript.

Cada aula contém:
- `index.html`: Página HTML principal com o exemplo.
- `main.js`: Arquivo JavaScript com o código da lição (quando aplicável).
- `exercícios/`: Pasta com exercícios adicionais (em algumas aulas).

## Como Usar

1. Clone ou baixe este repositório.
2. Abra qualquer arquivo `index.html` em um navegador web (como Chrome, Firefox ou Edge).
3. Abra o console do navegador (F12 > Console) para ver as saídas dos scripts.
4. Para exercícios, siga as instruções nos arquivos correspondentes.

## Pré-requisitos

- Um navegador web moderno com suporte a JavaScript.
